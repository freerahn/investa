# 블로그 사용법

## 1. 의존성 설치

먼저 필요한 패키지를 설치해야 합니다:

```powershell
npm install
```

## 2. 서버 실행

서버를 실행합니다:

```powershell
npm start
```

서버가 실행되면 다음과 같은 메시지가 표시됩니다:
```
블로그 서버가 http://localhost:3000 에서 실행 중입니다.
```

## 3. 브라우저에서 접속

- **메인 페이지**: http://localhost:3000
- **관리자 페이지**: http://localhost:3000/admin.html

## 4. 글 작성하기

1. 관리자 페이지(http://localhost:3000/admin.html)로 이동
2. 제목과 내용을 입력
3. "작성" 버튼 클릭

## 문제 해결

### 글 작성이 안되는 경우

1. **서버가 실행 중인지 확인**
   - 터미널에서 `npm start` 실행
   - "블로그 서버가 http://localhost:3000 에서 실행 중입니다." 메시지 확인

2. **브라우저 개발자 도구 확인**
   - F12 키를 눌러 개발자 도구 열기
   - Console 탭에서 에러 메시지 확인
   - Network 탭에서 API 요청 상태 확인

3. **서버 콘솔 확인**
   - 터미널에서 에러 메시지 확인
   - 글 작성 시 "글 작성 요청", "파일 저장 경로" 등의 로그 확인

4. **posts 디렉토리 확인**
   - `posts` 폴더가 생성되었는지 확인
   - 파일 쓰기 권한이 있는지 확인

## 로컬 PC에서 외부 접속 허용

다른 사람들이 접속하려면 서버를 외부에서 접근 가능하도록 설정해야 합니다:

### 방법 1: 같은 네트워크에서 IP 주소로 접속

1. IP 주소 확인:
```powershell
ipconfig
```
IPv4 주소를 찾으세요 (예: 192.168.0.100)

2. server.js 수정:
```javascript
app.listen(PORT, '0.0.0.0', () => {
  // ...
});
```

3. 방화벽에서 포트 3000 허용:
   - Windows 방화벽 설정에서 포트 3000 인바운드 규칙 추가

4. 다른 사람들은 `http://[당신의IP]:3000` 으로 접속

### 방법 2: 인터넷을 통해 접속 (고급)

- ngrok, cloudflare tunnel 등을 사용하여 외부 접속 가능하게 설정
